Function in the library



tekpixel



The function tekpixel change the color of one pixel.
The function is define as below:

void      tekpixel(t_bunny_pixelarray *pix, t_bunny_position *pos, t_color *color);

The function tekpixel can change the color only if the pixel is on the pixelarray.
If not, it will do nothing.



tekgui_load



The tekgui_load function load all information from the ini file and complete the structure.
You must calll this function first to be able to use all other.
tekgui_load is define as below:

t_tekgui     tekgui_load(const char *filename);


If an error is found, the function will return NULL and print an error.



tekgui_display



The function tekgui_display write all the information from the structure to the pixelarray.
Warning : tekgui_display will not display the pixelarray, she will just complete it
If you want to display the window, you must call the fonction display_window
tekgui_display is define as below:

void       tekgui_display(t_bunny_pixelarray *pix, t_tekgui *tekgui);



To do something more simply, u can use the function like the example below

tekgui_display(tekgui->pix, tekgui);



display_window



The function display_window print the window.

void      display_window(t_tekgui *tekgui);

This function print the pixelarray, but can not set it. To set the pixelarray, you must use tekgui_display



tekfunction



tekfunction take in parameter the name of a function and return the adresse of this function.
tekfunction is define as below

void      *tekfunction(const char *functname);



If the function does not exist or an error append, tekfunction will return NULL.



tektext



This function print a string.
tektext take in parameter two pixelarray and a string to print.
tektext is define as below :

void tektext(t_bunny_pixelarray *out, const char *str, t_bunny_pixelarray *font_png);



tekgetpixel



The function tekgetpixel take a pixelarray and a position of a pixel and return his color.
tekgetpixel is define as below :

t_color     tekgetpixel(t_bunny_pixelarray *, t_bunny_position *);



tekgui_stop



The function tekgui_stop free the memory.
The function must be called at the end of the program.
The function is define ad below :

void      tekgui_stop(t_tekgui *);



tekgui_max_ram



The function tekgui_max_ram take in parameter the ini file, and set the maximum ram as definie in the ini.
tekgui_max_ram is definie as below :

void    tekgui_max_ram(const char *name);

U must call the function before tekgui_load
